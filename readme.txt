# autoreg plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

autoreg plugin (v1.3.1) for B3

Installation:

1. Place the autoreg.py in your ../b3/extplugins and the 
autoreg.xml in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="autoreg" config="@b3/extplugins/conf/autoreg.xml"/>
